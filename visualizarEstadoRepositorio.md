### Resumen repositorio

Vemos un resumen de cómo está el archivo.

```sh 
$ git status
```

Vemos un resumen de como está el archivo en una línea.

```sh 
$ git status –s
```

Ver resumen de como está el archivo y la rama en la que estamos.

```sh 
$ git status –s –b
```

Sería lo mismo que:

```sh
$ git status -sb
```
